How to upload a file

To submit more than {{max_constraints}} constraints, write them in a text file and upload it in the drag and drop zone. You can use either minimization or maximization, but make sure to follow the syntax provided in the example below:

min: x + y
subject to:
  • constraint 1

  • ...
  • constraint n

subject to:
  • constraint 1
